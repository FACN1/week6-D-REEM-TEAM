### Code Styles
- **Indentation** => 2 spaces
  ```js
  function placeholder() {
    var variable1 = '';
  //
  }
  ```
- **Functions** =>
  - Client-side is `normal function`
    ```js
    var apple = function(size) {

    }
    ```

  - Server-side is `arrow function`

    ```js
    var apple = (size) => {

    }
    ```
